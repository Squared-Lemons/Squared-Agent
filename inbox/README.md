# Inbox

Drop ideas and feedback here for discussion and planning.

## Folders

### ideas/
Your ideas for improving this agent. Create a markdown file with:
- What you want to improve
- Why it matters
- Any initial thoughts

### from-projects/
Feedback from projects created with this agent. When running `/end-session` in a spawned project, copy the generated feedback here.

File naming: `YYYY-MM-DD-[project-name].md`

## What Happens Next

I'll review inbox items and create proposals in `suggestions/` when I see actionable improvements. We can then discuss and implement them together.

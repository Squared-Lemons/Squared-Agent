## Structure

Each client folder contains:
- `README.md` - Client overview, contacts, key info
- `research/` - Research findings and analysis
- `documents/` - PDFs, contracts, proposals
- `notes/` - Meeting notes, ideas, correspondence

## Adding a New Client

1. Copy `_TEMPLATE/` folder
2. Rename to client name (e.g., `Acme-Corp/`)
3. Fill in the README.md template
4. Add to the directory table above
